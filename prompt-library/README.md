# Prompt Library

AI prompt templates for each stage of the Community Data Lab research workflow. Designed for Claude Code, adaptable to any capable AI coding assistant.

## Organization

Prompts are organized by research workflow stage:

- `01-research-questions/` — Framing testable equity questions
- `02-data-sourcing/` — Finding and verifying public data sources
- `03-data-processing/` — Writing fetch and transform scripts
- `04-analysis/` — Running regressions, scoring, and outlier detection
- `05-output-production/` — Building viewers, briefs, and workbooks
- `06-maintenance/` — Refresh schedules, playbooks, memory architecture

## How to Use

Each prompt template includes:
- **Context block** — what to tell the AI about your project
- **Instruction block** — what to ask for
- **Constraints block** — quality guardrails (stdlib only, documented sources, etc.)
- **Example output** — what a good result looks like

Replace `[bracketed]` placeholders with your project-specific details.
